# 🐍 Snake Game (Python Turtle)

A classic Snake game built using Python Turtle graphics, demonstrating object-oriented programming and game logic.

---

## 🚀 Features

* 🎮 Classic snake gameplay
* 🍎 Food spawning and score tracking
* 💥 Collision detection (wall & self)
* ⏩ Increasing difficulty

---

## 🛠️ Tech Stack

* Python
* Turtle Graphics
* Object-Oriented Programming

---

## 📂 Project Structure

```
snake-game-python/
│
├── main.py
├── snake.py
├── food.py
├── scoreboard.py
```

---

## ⚙️ How to Run

1. Clone the repository
2. Run:

```
python main.py
```

---

## 🎮 Controls

* ↑ → Move Up
* ↓ → Move Down
* ← → Move Left
* → → Move Right
